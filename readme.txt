4.) the project includes following files:
	circuit design files (json files, pdf-file)
	gerber files for pcb-production
	different screenshots and pictures of pcb and pcb case
	bom (bill of material)
	stl file for 3d printable case
	skp (sketchup) design file for 3d printable case 
	gcode file for direct printing (material pla, 0.4mm nozzle, 0.2mm layer height, infill=20%, bed temp 60°, hotend temp 190°)
	pin assignment text file
	readme.txt - this file
